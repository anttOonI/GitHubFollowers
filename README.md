# GitHubFollowers
Проект выполненный по курсу SeanAllen

## Технологии, используемые в проекте

+ Архитектура MVC
+ Проектировние интерфейса в кодe
+ Кастомные UI элементы
+ UICollectionViewDiffableDataSource
+ Работа с API github при помощи URLSession
